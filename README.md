# 汇率转换工具
支持筹码、CNY、USD、BB 自动换算的静态工具页面。

## 汇率
1 BB = 0.02 USD  
1 USD = 6.9038 CNY  
1000 筹码 = 40 CNY

## 功能
筹码计算、自动换算、回车快捷、一键重置。

## 使用
直接打开 index.html 即可使用。

## 一、项目概述
开发一款**静态页面汇率转换工具**，支持筹码、人民币(CNY)、美元(USD)、BB 四者之间自动换算，界面美观、交互简洁，可直接在浏览器运行。

---

## 二、界面布局（固定顺序）
页面从上至下顺序：
1. 标题：汇率转换工具
2. **筹码计算模块**
3. CNY / ￥ 输入框
4. USD / $ 输入框
5. BB 输入框
6. 重置按钮
7. 说明信息区

---

## 三、核心汇率规则
1. **BB ↔ USD**
   - 1 BB = 0.02 USD
   - 换算比例：50 倍
2. **USD ↔ CNY**
   - 1 USD = 6.9038 CNY
3. **筹码 ↔ CNY**
   - 1000 筹码 = 40 元 CNY
   - 换算系数：0.04

---

## 四、筹码模块需求
1. 支持面值：**500、100、50、20、10**
2. 布局：每个面值**上方显示文字，下方输入框**
3. 输入框：**数字内容必须水平居中**
4. 最右侧显示：**筹码总值**（只读、禁用、不可编辑）
5. 计算公式：
   ```
   总额 = 500×a + 100×b + 50×c + 20×d + 10×e
   ```
6. 筹码仅用于计算 **CNY**
7. **输入筹码时，自动禁用 USD、BB 输入框**

---

## 五、计算逻辑需求
1. 输入任意一项，**自动计算其他所有项**
2. 所有结果**保留 4 位小数**
3. 支持 **Enter 回车键快捷计算**
4. 支持 **一键重置所有内容**
5. 空值 / 非数字自动按 0 处理

---

## 六、交互规则
1. 输入筹码 → 计算筹码总额 + CNY；**禁用 USD/BB**
2. 输入 CNY → 自动计算 USD、BB；**清空筹码**
3. 输入 USD → 自动计算 CNY、BB；**清空筹码**
4. 输入 BB → 自动计算 USD、CNY；**清空筹码**
5. 点击重置 → 清空所有输入、恢复可用状态

---
