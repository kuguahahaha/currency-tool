# 汇率转换工具
支持筹码、CNY、USD、BB 自动换算的静态工具页面。

## 汇率
1 BB = 0.02 USD  
1 USD = 6.9038 CNY  
1000 筹码 = 40 CNY

## 功能
筹码计算、自动换算、回车快捷、一键重置。

## 使用
直接打开 index.html 即可使用。

USD = CNY ÷ 6.9038
